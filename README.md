# LIE:BLOOMED

## 📖 Sinopse
**LIE:BLOOMED** é uma light novel que explora conflitos psicológicos, decisões difíceis e personagens marcados por seus próprios segredos.  
Em um mundo onde a verdade pode ser manipulada, cada escolha pode alterar completamente o destino dos envolvidos.  

A obra mistura mistério, drama e tensão narrativa, criando uma história intensa e cheia de reviravoltas.

## 🧠 Temas
